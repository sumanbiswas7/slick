# Close Button

```js
import { CloseButton } from "@sumanbiswas7/slick";

<CloseButton />;
```

### Props

| Name         | Type           | Description                                | Required |
| ------------ | -------------- | ------------------------------------------ | -------- |
| `background` | color          | Changes the background color of the button | No       |
| `hexBg`      | string [#fafa] | Gives a hex background color               | No       |
| `onClick`    | function       | Handle onClick event                       | No       |
| `color`      | string [#f2f2] | Change the cross color                     | No       |
| `size`       | size           | Changes the size of the button             | No       |
| `radius`     | radius         | Changes the border-radius of the button    | No       |
| `style`      | style object   | Addictional styles                         | No       |
