# CircleLoaderV2

```js
import { CircleLoaderV2 } from "sumanbiswas-slick";

function App() {
  return (
    <>
      <CircleLoaderV2 />;
    </>
  );
}
```

### Props

| Name    | Type           | Description                    | Required |
| ------- | -------------- | ------------------------------ | -------- |
| `size`  | size           | Changes the size of the Loader | No       |
| `color` | string [#f2f2] | Changes the loader color       | No       |
| `hexBg` | string [#fafa] | Gives the loader a hex color   | No       |
| `style` | style object   | Addictional styles             | No       |
