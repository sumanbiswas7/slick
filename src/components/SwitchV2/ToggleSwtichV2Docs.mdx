# ToggleSwitchV2

```js
import { ToggleSwitchV2 } from "@sumanbiswas7/slick";

const handleValueChange = (val: boolean) => {
  console.log(val);
};

<ToggleSwitchV2 value={handleValueChange} />;
```

### Props

| Name         | Type     | Description                    | Required |
| ------------ | -------- | ------------------------------ | -------- |
| `value`      | function | Dispaches checkbox check value | No       |
| `background` | string   | Background color               | No       |
| `hexBg`      | string   | Hex color for background       | No       |
| `size`       | number   | Changes the size of switch     | No       |
