# InputV2

```js
import { InputV2 } from "sumanbiswas-slick";
import { MdOutlineAlternateEmail } from "react-icons/md";

function App() {
  return (
    <>
      <Input
        placeholder="Your name"
        icon={<MdOutlineAccountCircle color="#CED4DA" />}
      />
    </>
  );
}
```

### Props

| Name           | Type         | Description                            | Required |
| -------------- | ------------ | -------------------------------------- | -------- |
| `icon`         | icon         | Input icon                             | Yes      |
| `inputStyle`   | style object | Styles for input                       | No       |
| `wrapperStyle` | style object | Styles for wrapper                     | No       |
| `accent`       | color        | Changes the outline color of the input | No       |
| `hexAccent`    | hex-color    | Changes the outline color of the input | No       |
| `radius`       | radius       | Changes the border-radius of the input | No       |
| `onChange`     | function     | Handle onChange event                  | No       |
| `placeholder`  | string       | Placeholder text                       | No       |
