# Input

```js
import { Input } from "@sumanbiswas7/slick";

function App() {
  return (
    <>
      <Input label="Name" placeholder="Your name" required />;
    </>
  );
}
```

### Props

| Name       | Type         | Description                            | Required |
| ---------- | ------------ | -------------------------------------- | -------- |
| `style`    | style object | Addictional styles                     | No       |
| `label`    | string       | Changes the label of the input         | No       |
| `accent`   | color        | Changes the outline color of the input | No       |
| `size`     | size         | Changes the size of the input          | No       |
| `radius`   | radius       | Changes the border-radius of the input | No       |
| `required` | boolean      | Gives a \* following the label         | No       |
| `onChange` | function     | Handle onChange event                  | No       |
